# Game Explorer 🎮  

A modern web app to explore games with features like sorting, filtering, dark mode, and responsive design. This is my practice project for React 18 Beginners, based on the course React 18 for Beginners by Code with Mosh. Available here: https://codewithmosh.com/p/ultimate-react-part1 

---

## Features ✨  
### Core Features  
- **Game Listings**: Display game cards with images, scores, and platform icons.  
- **Search Functionality**: Search for your favorite games easily.  
- **Genre Filtering**: Filter games by genres with a bold highlight on the selected genre.  
- **Platform Filtering**: Use the platform selector to refine your search.  
- **Sorting Options**: Dynamically sort games by different criteria
